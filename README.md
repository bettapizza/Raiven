# Lever-Inspired Jekyll Theme

A free Jekyll portfolio starter inspired by the layout and feel of the Jekyll Lever demo.

## Run locally

```bash
bundle install
bundle exec jekyll serve
```

## Customize

Edit these files first:

- `_config.yml` for name, intro, links, and footer info
- `_data/projects.yml` for homepage and work page cards
- `about.md` for the about page content
- `assets/css/style.css` for colors, spacing, and typography

## Deploy to GitHub Pages

1. Push this folder to a GitHub repo
2. In GitHub, open **Settings → Pages**
3. Set the source to your default branch and root folder
4. Save and wait for the site to publish

## Notes

This theme is an original recreation inspired by the referenced demo’s overall structure and styling. It does not include the original premium theme’s source code, proprietary assets, or branding.
